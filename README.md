## 编码规范
1. 全部使用简体字。
1. 易经内容包括卦辞和爻辞，不使用标点。
1. 爻辞后引用二十四史，标题为出处，正文分段使用标点。引用尽可能保持故事的完整性，以段为基本单位，多条引用要保证有序。
1. 不应有除了易经和二十四史以外的废话。
